# Deluge LTConfig Tuner

Professional Deluge LTConfig Tuner optimized for Private Tracker Racing.

## Features

- Optimized for Private Tracker Racing
- Embedded LTConfig
- Automatic Backup
- Automatic Permission Fix
- One Command Installation
- Debian 10 / 11 / 12 / 13 Support

## Installation

```bash
bash <(wget -qO- https://raw.githubusercontent.com/race-in/Deluge-LTConfig-Tuner/main/tune.sh)
```
